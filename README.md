Sudoku Generator
by Sunay Jain
This is a Sudoku Generator that can generate boards that are all solvable with one unique solution.
You run it by just calling the main function and answering the questions printed in the terminal.
The sudoku is generated by a recursive backtracking algorithm (technically 2 one to generate a solved board and one to count the number of valid board states) and is guarunteed to have one unique solution.
